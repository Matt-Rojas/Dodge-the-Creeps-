# Dodge the Creeps!
This project was created from the Godot documents on creating your first 2d game.
The only thing that is different is that I added controller support and added a mute button to mute the music.

# Instruction
1. Click the .zip file.
2. Since the file is too big, either click "View Raw"(it should download automatically) or the Download icon on the right side.
3. Extract the "Dodge_the_Creeps!.zip" file.
4. Once extracted, open the folder and open "Dodge the Creeps!.exe"
5. Have FUN!
